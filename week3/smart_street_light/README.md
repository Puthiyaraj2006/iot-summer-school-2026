# Q23 - Smart Street Light

## Author
Puthiyaraj P

## Components Used
- Arduino Uno
- LDR Sensor
- PIR Motion Sensor
- LED
- 220Ω Resistor

## Working
- Daylight: LED OFF
- Night: Motion detection enabled
- Motion detected: LED ON (100%) for 30 seconds
- No motion after 30 seconds: LED DIM (20%)

## Output
Events are displayed on the Serial Monitor with timestamps.
