# Q25 - Servo Motor Sweep

## Author
Puthiyaraj P

## Components Used
- Arduino Uno
- SG90 Servo Motor
- Potentiometer
- Push Button

## Working
- Potentiometer controls the servo angle from 0° to 180°.
- Current servo angle is displayed on the Serial Monitor.
- Pressing the push button makes the servo perform one complete sweep (0° → 180° → 0°).
