# Q22 - Ultrasonic Parking Sensor

## Author
Puthiyaraj

## Description
Measures distance using the HC-SR04 ultrasonic sensor and controls LEDs and buzzer based on parking distance.

## Components Used
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- Red LED
- Yellow LED
- Piezo Buzzer
- 220Ω Resistors

## Formula
distance = (duration * 0.034) / 2

## Bonus
Uses millis() for non-blocking timing.
