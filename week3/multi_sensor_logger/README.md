# Q24 - Multi Sensor Data Logger

## Author
Puthiyaraj P

## Components Used
- Arduino Uno
- DHT22
- HC-SR04
- LDR Sensor

## Working
- Reads Temperature and Humidity
- Reads Light Level and converts to %
- Measures Distance using HC-SR04
- Displays sensor data every 5 seconds
