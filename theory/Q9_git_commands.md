# Q9 - Difference between git pull, git fetch, and git clone

## git clone
Purpose: Downloads the entire repository to your computer.
Use: When downloading a repository for the first time.

## git fetch
Purpose: Downloads the latest changes without merging them.
Use: When you want to check updates before merging.

## git pull
Purpose: Downloads the latest changes and merges them into your local branch.
Use: When you want to update your local repository.

| Command | Purpose | When to Use |
|---------|---------|-------------|
| git clone | Downloads the repository | First time |
| git fetch | Downloads changes without merging | Check updates |
| git pull | Downloads and merges changes | Update local repository |
